This year marked a significant transition in my career as I expanded my skillset to become a full-stack developer. The journey involved learning server-side technologies, database design, and cloud infrastructure.

## What I learned

- **Backend Development**: Built RESTful APIs with Node.js and Express
- **Database Design**: Learned SQL, normalization, and query optimization
- **DevOps Basics**: Containerization with Docker and deployment on AWS
- **System Architecture**: Understanding scalability, caching, and microservices

## Challenges faced

The biggest challenge was shifting my mindset from purely visual, user-facing work to thinking about data flow, security, and server performance. Debugging backend issues required a different approach than frontend bugs.

## Impact

This transition opened up new opportunities and gave me a more holistic understanding of web applications. I can now build complete features from database to UI.
